1. 没有锁保护

有如下结构体:

struct your_obj {
    struct hlist_node obj_node_hlist;
    struct rcu_head rcu_head;   
    atomic_t refcnt;
    int id;
};

错误写法:

hlist_del_init_rcu(&your_obj->obj_node_hlist);

正确写法:

spin_lock(&obj_hash_lock[hash]);
hlist_del_init_rcu(&your_obj->obj_node_hlist);
spin_unlock(&obj_hash_lock[hash]);
